## Projeto 1: Dashboard de Vendas Globais

Desenvolvimento de um dashboard interativo em Microsoft Power BI utilizando um dataset fictício de vendas de uma empresa com atuação global. O projeto teve como objetivo transformar dados brutos em informações visuais e estratégicas para apoio à tomada de decisão, contemplando análises de desempenho comercial e indicadores de vendas.

### O dashboard apresenta:
- Valor total vendido;
- Quantidade de vendas por categoria de produto;
- Vendas por país considerando prioridade de entrega;
- Média de desconto por subcategoria;
- Países com maior média de valor de venda, exibidos em mapa geográfico.
